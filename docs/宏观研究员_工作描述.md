# 宏观研究员

- **薪资：** 30-45K·14薪
- **地点：** 上海
- **经验要求：** 经验不限
- **学历要求：** 硕士

## 职位标签

- 卖方行研经验
- 宏观经济研究
- 掌握 Python/R 语言/Matlab 等一种或多种数据科学技能
- 期货研究
- 行业及金融复合背景
- 擅长证券研究
- 擅长宏观经济/政策研究

## 职责描述

1. **宏观数据跟踪与趋势推导：** 负责跟踪全球及国内宏观经济数据的变化，准确描述当前宏观状态，推导边际变化趋势，并基于此提出前瞻性的投资或应对策略。
2. **黑天鹅与灰犀牛事件前瞻推演：** 针对全球地缘政治及宏观经济的重大极端事件进行跟踪以及前瞻预判。
3. **多智能体博弈系统研发：** 利用大模型应用技术，优化专门的垂类大模型与多 Agent 博弈模式，通过模拟不同利益主体的对抗与辩论，实现复杂宏观事件的事前预测。
4. **范式模型化：** 参与构建宏观预测模型（包含指标与事件）；跟踪并系统化拆解行业顶尖独立智库的研究框架（国内和国外），将其转化为可验证的量化模型。

## 任职要求

1. **硕士及以上学历，** 经济学、政治学、国际关系、计算机科学、数据科学、应用数学或计算机金融等相关复合背景优先。
2. **专业特质与意识：** 具备较强的分析、判断、研究能力和敏锐的市场意识；对“前瞻预测”有深厚学术兴趣，具备良好的概率思维与严谨的批判性思维。
3. **技术工程能力：** 熟练掌握 Python 编程语言，熟悉大语言模型（LLM）提示词工程；能熟练运用或快速掌握主流 Multi-Agent 框架（如 LangChain、AutoGen、CrewAI 等）进行流程构建。
4. **行业经验：** 有证券、期货、银行等金融行业相关工作经历者优先。拥有金融宏观量化研究经验，或在 Multi-Agent 模拟、复杂系统建模、预测市场等领域有实际项目或论文发表者优先。
